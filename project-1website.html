Reflection

About me: I am a coder from Sydney. This is my first project. Building my first website has been a rollercoaster full of frustrations and triumphs. When I first started this project, I thought it was a simple task, but as I gradually learnt how both HTML and CSS work, I realised building a proper functioning website was actually a difficult task. The learning curve was steep, particularly when I started to learn about HTML and Flexbox. I spent numerous hours both at home and school trying to figure out how HTML worked and how to code so that the website adjusted to the device screen. In my journey, I made COUNTLESS errors and mistakes, often on the simplest things, such as a typo or missing a semicolon. However, these mistakes taught me how important it was to maintain focus when building a website. I learnt a lot about both HTML and CSS from softwares such as Github and FreeCodeCamp. I learnt from the mistakes I made and successfully implemented my HTML. I started using CSS to decorate my website with background colours, text colours, and even images and videos. However, the real turning point was when I stopped trying to fix or debug every mistake and started embracing my errors. I found useful code or fixes from places like Github and Reddit and this made me realise that no coders are perfect and many of them get help or inspiration from communities like Github and FreeCodeCamp. Overall, this project was a very fun experience and helped me learn what being a developer actually feels like and how incredible coding really is.
